
# SyntaxWear

SyntaxWear is a modern, stylish e-commerce platform dedicated to providing high-quality sneakers and footwear. This project is built with HTML and CSS, focusing on a clean, responsive design to ensure a seamless user experience across all devices.

## Project Structure

The project is organized into the following directories and files:

- **index.html**: The main entry point of the website, containing the structure and content of the homepage.
- **css/**: This directory holds all the stylesheets for the project.
  - **reset.css**: Resets default browser styles for consistency.
  - **variables.css**: Defines CSS variables for colors, fonts, and other global values.
  - **base.css**: Contains the base styles for the entire website.
  - **components/**: Includes component-specific styles for modularity.
    - **header.css**: Styles for the header section.
    - **hero.css**: Styles for the hero section.
    - **product-category.css**: Styles for the product category cards.
    - **product-grid.css**: Styles for the featured products grid.
    - **footer.css**: Styles for the footer section.
- **images/**: This directory contains all the images used in the project.
  - **banners/**: Images for banners and hero sections.
  - **favicons/**: Favicon files.
  - **icons/**: SVG icons for various UI elements.
  - **logo/**: The website's logo.
  - **products/**: Images of the products.

## Key Features

- **Responsive Design**: The website is fully responsive, adapting to different screen sizes from mobile to desktop.
- **Modern UI/UX**: A clean and modern user interface with a focus on user experience.
- **CSS Modularity**: The CSS is organized into components, making it easy to maintain and scale.
- **Product Showcase**: The homepage features a hero section, category cards, and a grid of featured products.
- **Newsletter Subscription**: A newsletter subscription form in the footer to keep users updated.

## Getting Started

To get a local copy up and running, follow these simple steps:

1. **Clone the repository**:
   ```sh
   git clone https://github.com/your-username/ecommerce-syntaxwear.git
   ```
2. **Navigate to the project directory**:
   ```sh
   cd ecommerce-syntaxwear
   ```
3. **Open `index.html` in your browser**:
   You can do this by double-clicking the file or by using a live server extension in your code editor.
